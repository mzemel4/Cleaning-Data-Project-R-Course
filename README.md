### Creating Final Project Tidy Data Set

###  Description
This repository contains the files needed to create the tidy data set *means_data.txt*.  This tidy data set contains summary measures from the raw accelerometer data from the UCI HAR project. Details about this project can be found at this [link](http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones)

### Files
This repository contains the following files

* tidy data set: **data/means_data.txt**

* Codebook for means_data tidy data set: **codebook.md**

* R code script for processing raw data and creating the tidy data set *means_data.txt*: **run_analysis.r**

* Raw Data Files from UCI HAR project: **data/extract/UCI HAR DATASET/**:

    *  **features.txt**
    * **features_info.txt**
    * **activity_labels.txt**
    * **test/X_test.txt**
    * **test/subject_test.txt**
    * **test/y_test.txt**
    * **train/X_train.txt**
    * **train/subject_train.txt**
    * **train/y_train.txt**
